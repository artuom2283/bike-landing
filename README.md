## 📦 Getting Started
Follow these steps to set up the project locally:

### 1️⃣ Clone the repository
```sh
git clone https://github.com/artuom2283/bike-landing.git
cd bike-landing
```

### 2️⃣ Install dependencies
```sh
npm install
# or
yarn install
```

### 3️⃣ Run the project locally
```sh
npm start
# or
yarn start
```
